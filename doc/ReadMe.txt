
实现了雪人基本功能以及键盘鼠标漫游，天空盒，地形，雪花粒子效果。

========================================================================
    WIN32 APPLICATION : 代码结构说明
========================================================================

main.cpp文件：
创建窗体。

directx_common.h文件：
实现在SnowMan3D.cpp中。加载静态库，及公共头文件，及实现工具函数。

SnowMan3D类：
初始化DirectX3D，在SnowMan3D::Render()完成所有的绘制。

SnowMan类：
画雪人。

SkyBox类：
加载天空盒。

SnowTerrian类：
从龙书复制，加载雪地地形。

SnowParticle类：
从龙书复制，下雪。

DirectXInput类：
获取鼠标键盘输入。

=========================================================================
控制说明：

W A S D: 前后左右移动。
R F:  上升下降。
上下： 上下旋转视角。
左右: 左右旋转视角。
鼠标： 按住左键，前后拖，左右拖。滚动调高视角。
空格： 恢复开始的视角。
